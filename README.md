Ushbu veb-loyiha maktab o'qituvchilari va rahbariyatini zamonaviy interaktiv ko'rinishda tanishtirish uchun yaratilgan raqamli albomdir. Loyiha o'zining "Luxury Gold" dizayni va dinamik elementlari bilan ajralib turadi.

---

## 🌟 Asosiy Xususiyatlar

* **🎬 Video Background:** Orqa fonda YouTube videosi orqali yaratilgan dinamik atmosfer.
* **✨ Modal System:** O'qituvchi ustiga bosilganda uning rasmi va ma'lumotlarini kattalashtirib ko'rsatuvchi interaktiv oyna.
* **📱 Full Responsive:** Smartfondan tortib 4K monitorlargacha moslashuvchan (responsive) dizayn.
* **🎨 Glassmorphism & Gold Theme:** Shaffof elementlar va oltin rangli hoshiyalar bilan boyitilgan premium interfeys.
* **⚡ Smooth Animations:** Kartochkalarning hover effektlari va modalning silliq ochilishi.

---

## 🛠 Texnik Stack

* **Frontend:** Sof HTML5, CSS3 va JavaScript (Vanilla JS).
* **Grid System:** Murakkab joylashuvlarni boshqarish uchun `CSS Grid` ishlatilgan.
* **Background Integration:** YouTube IFrame API orqali fondagi videoni boshqarish.
* **UX/UI:** Georgia va serif shriftlar uyg'unligi orqali akademik muhit yaratilgan.
